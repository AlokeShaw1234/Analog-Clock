🕒 Analog Clock Web App

🚀 Overview : 

A real-time Analog Clock built using HTML, CSS, and JavaScript. This project focuses on precise time synchronization and smooth UI rendering using core JavaScript concepts.

✨ Features :

- Real-time clock with accurate time updates
- Smooth rotation of hour, minute, and second hands
- Minimal and clean UI design
- Fully responsive layout

🛠️ Tech Stack :

- HTML5
- CSS3 (Transforms & Positioning)
- JavaScript (Date API, DOM Manipulation)

  
🔗 Live Demo :

👉 https://analog-clock-one-sepia.vercel.app/

🎯 Why this project matters :

This project demonstrates my ability to work with real-time data, DOM updates, and CSS transformations to build interactive UI components without using external libraries.

💡 What I Learned : 

- Working with JavaScript Date object.
- Applying CSS transforms for dynamic movement.
- Handling real-time UI updates.
