Readme : 

                                           Analog Clock Web App

Introduction :  A real-time Analog Clock built using HTML, CSS, and JavaScript. This project focuses on precise time synchronization and                  smooth UI rendering using core JavaScript concepts.

Tech Stack :

- HTML5
- CSS3 
- JavaScript

 Features :

- Real-time clock with accurate time updates
- Smooth rotation of hour, minute, and second hands
- Minimal and clean User-friendly interface.
- Fully responsive design for differenet screen sizes.

  
 Live Demo link : https://analog-clock-one-sepia.vercel.app/

Why this project matters :

This project demonstrates my ability to work with real-time data, Dynamic DOM updates, and CSS transformations to build interactive UI components without using external libraries.

Key Learnings : 

- Working with JavaScript Date object.
- Applying CSS transforms for dynamic movement.
- Handling real-time UI updates.
